# A-1: Solve DC Power Flow

## Result: PASS

## Approach

Loaded IEEE 39-bus network via `matpowercaseframes.CaseFrames` into a pypower `ppc` dict,
then imported into PyPSA via `net.import_from_pypower_ppc(ppc)`. Solved DC power flow
(linear power flow) with `net.lpf()` -- no solver selection needed, this is a direct
linear solve.

Extracted results from PyPSA's time-series DataFrames:
- `net.buses_t.v_ang` -- voltage angles per bus per snapshot
- `net.buses_t.p` -- nodal power injections per bus per snapshot
- `net.lines_t.p0` / `net.lines_t.p1` -- line active power flows at each end
- `net.transformers_t.p0` / `net.transformers_t.p1` -- transformer flows
