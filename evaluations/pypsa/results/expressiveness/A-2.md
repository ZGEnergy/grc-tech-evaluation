## Result: PASS

## Approach

Loaded IEEE 39-bus network via the standard `matpowercaseframes` + `import_from_pypower_ppc`
path. Solved AC power flow using PyPSA's built-in Newton-Raphson solver via `net.pf()`.

Following the convergence protocol:
1. **Flat start** (default): all voltage magnitudes at their setpoints, angles at 0.0 rad.
2. Result: **Converged on flat start** -- no DC warm start fallback was needed.

Extracted structured outputs:
- `net.buses_t.v_mag_pu` -- bus voltage magnitudes (p.u.)
- `net.buses_t.v_ang` -- bus voltage angles (rad)
- `net.lines_t.p0/p1` -- line active power flows
- `net.lines_t.q0/q1` -- line reactive power flows
- Losses computed as `p0 + p1` per branch
