# F-1: Core license

## Finding

## Evidence
- PyPI project page declares MIT License
- GitHub repository LICENSE file confirms MIT
- `pypsa` package metadata classifier: `License :: OSI Approved :: MIT License`
- Installed package `License-Expression` field contains the full MIT license text
- 104 contributors listed on GitHub, indicating broad community stewardship
- SPDX headers (`SPDX-License-Identifier: MIT`) present in source files

## Implications
