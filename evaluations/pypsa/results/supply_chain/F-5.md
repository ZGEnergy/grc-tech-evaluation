# F-5: Code inspectability (DCPF trace)

## Finding
The entire DCPF (Linear Power Flow) path from `net.lpf()` to the sparse linear solve is fully traceable through pure Python code. No opaque steps. The mathematical operations are explicit and readable.

## Evidence

### Complete call chain for `net.lpf()`

1. **`NetworkPowerFlowMixin.lpf()`** (`pypsa/network/power_flow.py:805`)
   - Entry point. Converts snapshots, delegates to `_network_prepare_and_run_pf(n, sns, skip_pre, linear=True)`.

2. **`_network_prepare_and_run_pf()`** (`pypsa/network/power_flow.py:143`)
   - Orchestrator. Calls `n.calculate_dependent_values()`, identifies sub-networks, then calls `sub_network.lpf()` for each.

3. **`NetworkPowerFlowMixin.calculate_dependent_values()`** (`power_flow.py:732`)
   - Computes per-unit impedances: `x_pu = x / v_nom^2`, `r_pu = r / s_nom` for transformers.
   - Pure pandas arithmetic on component DataFrames. Fully readable.

4. **`SubNetworkPowerFlowMixin.find_bus_controls()`** (`power_flow.py:1286`)
   - Identifies slack bus and generator for each sub-network.

5. **`SubNetworkPowerFlowMixin.calculate_B_H()`** (`power_flow.py:1079`)
   - **Core matrix construction.** Comments note it "leans heavily on pypower.makeBdc".
   - Computes branch susceptances: `b = 1/x_pu_eff`
   - Builds diagonal susceptance matrix `b_diag` (scipy sparse)
   - Gets incidence matrix `K` from `self.incidence_matrix()`
   - Computes transfer matrix: `H = b_diag * K^T`
   - Computes weighted Laplacian: `B = K * H`
   - Handles phase-shift transformers explicitly

6. **`SubNetworkPowerFlowMixin.lpf()`** (`power_flow.py:1746`)
   - **Core solve.** Aggregates power injections per bus from all one-port and controllable-branch components.
   - Calls `scipy.sparse.linalg.spsolve(B[1:,1:], p[:,1:].T)` for voltage angle differences.
   - Computes branch flows: `flows = v_diff * H^T + p_branch_shift`
   - Assigns results back to component DataFrames.
   - Handles slack bus power balance.
