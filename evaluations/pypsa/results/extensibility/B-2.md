# B-2: Graph access — BFS to depth 3

## Result: PASS

## Approach

PyPSA provides a documented public method `Network.graph()` that returns a NetworkX
`OrderedGraph` (a `MultiGraph` subclass). This is a first-class API — no workarounds,
no internal access, no conversion scaffolding needed.

From the exported NetworkX graph, standard NetworkX algorithms work directly. The test
uses `nx.bfs_edges(G, source, depth_limit=3)` and `nx.bfs_layers(G, source)` to
perform BFS to depth 3 from bus "1".

The graph includes all buses as nodes and all branches (lines + transformers) as edges,
with edges keyed by `(component_type, branch_name)` tuples — preserving full provenance
back to the PyPSA model.

```python
G = net.graph()                                    # OrderedGraph (NetworkX MultiGraph)
bfs_edges = list(nx.bfs_edges(G, "1", depth_limit=3))
```
