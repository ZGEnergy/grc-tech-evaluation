# C-1: DCPF at Scale (MEDIUM -- 10,000-bus network)

## Summary

PyPSA successfully solved DC power flow (linear power flow) on the 10,000-bus
ACTIVSg10k network in 6.69 seconds total (6.37s solve, 0.32s load).

## Method

- Loaded MATPOWER case via `matpowercaseframes` + `import_from_pypower_ppc`
- Used `overwrite_zero_s_nom=True` for zero-rated branches (sets to 1.0 MW)
- Solved with `net.lpf()` (sparse LU factorization)

## Observations

- The DCPF solve is dominated by sparse matrix factorization and back-substitution.
  PyPSA handles the 10k-bus network without difficulty.
- Memory usage is moderate at ~2.4 GB for a 10k-bus network.
- The solve time (~6s) is reasonable for a single-snapshot linear power flow on
  a network of this size.
