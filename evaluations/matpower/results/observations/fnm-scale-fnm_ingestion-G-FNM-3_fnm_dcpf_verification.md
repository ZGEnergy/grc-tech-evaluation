# Observation: MATPOWER DCPF scales efficiently to 27,862-bus FNM

## Finding

MATPOWER solves DCPF on the 27,862-bus FNM main island network in 0.217
seconds (Octave, single-threaded), with peak RSS of 1.9 MB. This is the
fastest DCPF solve time observed across all evaluated tools on this network.

## Context

G-FNM-3 loaded the pre-cleaned FNM case via `loadcase()` and solved DCPF
using `rundcpf()`. The solve produced float64-noise-level deviations against
the reference solution (max VA deviation 5.0e-9 deg, max branch flow
deviation 5.0e-7 pct). Both solutions were generated by the same MATPOWER
version from the same `.mat` file, confirming deterministic reproduction.

The v11 bus injection power balance check confirmed max mismatch of
6.2e-9 MW across all 27,862 non-excluded buses.

## Implications

For scalability (Suite C) assessment: MATPOWER's DCPF performance at FNM
scale (LARGE tier, ~28k buses) is well within any reasonable time budget.
The sub-second solve time suggests MATPOWER could handle repeated DCPF
solves (e.g., contingency sweeps, scenario analysis) efficiently at this
scale.
