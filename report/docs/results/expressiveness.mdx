# Expressiveness

## Overview

Expressiveness measures the breadth of power system analyses a tool can perform natively or with minimal extension. It is the **highest priority** criterion in the rubric, reflecting the core requirement that a selected tool must cover the analysis types ZGE needs without rewriting fundamental algorithms. The test suites A-1 through A-11 map to this criterion, spanning DC power flow (DCPF), AC power flow (ACPF), DC/AC optimal power flow (DCOPF/ACOPF), security-constrained variants (SCOPF, SCUC, SCED), and stochastic optimization.

Each tool was tested at three network scales — TINY (5-bus), SMALL (IEEE 118-bus), and MEDIUM (ACTIVSg10k) — to expose both capability gaps and scale-dependent failures. Tools that pass at TINY but fail at SMALL or MEDIUM reveal API limitations that would surface in production use.

## Tool Results

<details className="eval-details" open>
<summary>

### PyPSA — <span className="grade-b-plus">B+</span> (#1)

</summary>

PyPSA demonstrates strong native support for most target problem types including DCPF, ACPF, DCOPF, and SCED. It provides a clean high-level API through linopy for OPF formulations and natively supports SCOPF via iterative contingency screening. However, SCUC requires significant workaround effort and stochastic optimization is documented but non-functional (probe-002).

ACPF convergence on MEDIUM (ACTIVSg10k) was debunked by probe-001: the solver reports convergence with 0 NR iterations and 83% of buses at flat start. This changed A-2 MEDIUM from pass to qualified\_pass but did not affect the overall B+ grade.

<summary>

### PowerModels — <span className="grade-b">B</span> (#2)

</summary>

PowerModels provides the most mathematically rigorous OPF formulations among all evaluated tools, with native support for multiple AC OPF relaxations (SOC, QC, SDP) and SCOPF. Its JuMP-based architecture makes custom constraint injection straightforward. However, SCUC/SCED are not natively supported (requires PowerSimulations), and the Julia startup overhead impacts workflow iteration speed.
