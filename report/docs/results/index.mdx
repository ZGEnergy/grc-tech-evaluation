# Results Overview

The Results section presents the complete evaluation findings for all six power-system modeling tools across seven rubric dimensions. Six criterion sub-pages — [Expressiveness](./expressiveness), [Extensibility](./extensibility), [Scalability](./scalability), [Accessibility](./accessibility), [Maturity](./maturity), and [Supply Chain](./supply-chain) — each follow a uniform template with per-tool collapsible sections containing letter grades, evidence tables, and analytical narratives. Three cross-cutting pages complement the criterion deep-dives: [Head-to-Head](./head-to-head) compares tool capabilities on Phase 2-critical features, [Sweep Findings](./sweep-findings) documents systemic protocol themes, and [Probe Results](./probe-results) addresses specific disputed claims.

Each criterion page is organized identically: six collapsible tool sections (ordered by final rank), each containing a grade summary, a pass/fail evidence table covering all test IDs at three network scales (TINY, SMALL, MEDIUM), and a narrative explaining the grade rationale with references to specific test outcomes. Grades use a letter scale from A through F, displayed with color-coded badges. MATPOWER appears on every criterion page as a reference-only benchmark — it is not ranked due to its MATLAB licensing requirement, but its results provide a calibration baseline for the other five tools.

Grades were derived from a standardized evaluation protocol that executed six test suites (A through F) across three network scales (IEEE 39-bus, ACTIVSg 2000-bus, and ACTIVSg 10,000-bus). Each test produced a structured result with pass/fail status, wall-clock timing, and diagnostic notes. Letter grades aggregate these per-test outcomes within each criterion, weighting MEDIUM-scale results most heavily to reflect production relevance. The protocol v10 revision incorporated findings from 18 targeted probes and 13 cross-cutting themes identified during the v4-to-v5 sweep, along with subsequent principal review corrections, to ensure grading consistency.

PyPSA's #1 ranking is robust to most alternative criterion weighting schemes. The table below presents four scenarios — the baseline lexicographic priority ordering used in the evaluation, plus three alternative schemes. PyPSA holds the top position in three of four scenarios; the sole exception is Scenario 3 (Scalability as top priority), where GridCal takes #1. MATPOWER is excluded from ranking per the rubric's reference-only designation.

## Navigation Guide

### Criterion Sub-Pages

- **[Expressiveness](./expressiveness)** — Covers Suite A tests: DC/AC power flow, OPF variants, unit commitment, and SCOPF. The highest-priority criterion in the lexicographic ranking.
- **[Extensibility](./extensibility)** — Covers Suite B tests: custom constraints, solver swapping, PTDF/LODF extraction, and API depth for programmatic model manipulation.
- **[Scalability](./scalability)** — Covers Suite C tests: performance benchmarks across three network scales, measuring wall-clock time, memory usage, and convergence reliability.
- **[Accessibility](./accessibility)** — Covers Suite D tests: documentation quality, installation experience, error messages, and onboarding friction for new users.
- **[Maturity](./maturity)** — Covers Suite E tests: community health metrics, release cadence, CI/CD practices, and long-term maintenance indicators.
- **[Supply Chain](./supply-chain)** — Covers Suite F (gate) tests: licensing compatibility, dependency audit, and security posture. This criterion serves as a pass/fail gate before detailed evaluation.

### Cross-Cutting Pages

- **[Head-to-Head](./head-to-head)** — Direct 6-capability comparison between the top tools for Phase 2-critical features including SCOPF, stochastic dispatch, and custom constraint injection.
- **[Sweep Findings](./sweep-findings)** — Documents 13 cross-cutting themes identified during the v4-to-v5 protocol revision, covering network insufficiency, scoring inconsistency, and protocol gaps.
- **[Probe Results](./probe-results)** — Presents 18 targeted probes that verified extraordinary claims, including 4 debunked findings with full methodology and grade impact analysis.
