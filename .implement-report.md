## Implementation Details

### validate_charts.py

- `validate_manifest()` — Validates manifest JSON against files on disk, detects missing and extra chart files
- `validate_svg_file()` — Parses XML and checks for `<svg>` root element (with/without namespace)
- `validate_png_file()` — Checks first 8 bytes against PNG magic number signature
- `validate_all_chart_files()` — Validates format of every file referenced in manifest
- `validate_manifest_schema()` — Checks required top-level key, entry fields, and file format keys
- `run_charts_validate()` — CLI entry point for `make charts-validate`

### Integration Tests

Session-scoped fixture runs full pipeline with mocked `write_image` (stub SVG/PNG
content), avoiding Chrome/Kaleido dependency while validating the complete pipeline
flow including renderer registration, data loading, chart generation, export, and
manifest writing.
