# Implementation Report: PRD 06/03 — Version Stamp & Placeholder Validator

## Summary

Implemented `report/scripts/validate_content.py`, a post-build validation script
that checks every HTML doc page in the Docusaurus build output for:

1. Protocol version string (extracted from `docusaurus.config.js` `customFields.protocolVersion`)
2. "Last updated" timestamp
3. Absence of rendered `<Placeholder />` component output

Supports `--allow-placeholders` flag for permissive mode during development.
