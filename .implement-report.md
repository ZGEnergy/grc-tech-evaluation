## Design Decisions

- Field coverage gap table uses approximate counts with `~` prefix for MATPOWER .m coverage
  because the exact count depends on tool-specific .m format interpretation
- Record type gap table includes 6 record types covering all non-empty FNM types with
  degraded or absent .m representation
- Document quotes the rubric v2 grading note text closely to establish the precedent parallel
- Tests use H2 header splitting with keyword matching for section identification,
  allowing natural variation in section titles
