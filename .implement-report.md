# PRD 02/04 — Test Pass/Fail Matrix Renderer

## Summary

Implemented a heatmap-style test pass/fail matrix renderer that produces Plotly charts
from `TestResultsData`. Registered as `"matrix"` in the chart generation pipeline.

## Key Decisions

- **Single vs. split**: If total tests <= 40, renders one chart (`matrix_test-results`).
  If > 40, splits by suite with IDs like `matrix_test-results-{suite}`.
- **Color encoding**: 3-stop discrete colorscale — red (#f44336) for fail, gray (#9e9e9e)
  for skip, green (#4caf50) for pass.
- **Fallback**: When no individual test IDs exist, produces a bar-chart suite summary.
- **Suite separators**: Dashed white vertical lines at suite boundaries.
- **X-axis rotation**: -60 degrees for <= 20 tests, -90 for more.
