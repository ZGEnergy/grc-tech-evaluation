## Deviations from PRD

1. **Removed unused `import pytest`** from all three `test_gate.py` files. The PRD included `import pytest` at module level, but since no pytest markers or fixtures are used directly in the test file (the `data_dir` fixture comes from conftest.py), ruff flagged it as F401 (unused import). Removed to pass lint.

2. **Added `# noqa: F401`** to GridCal's `test_import` for `import VeraGridEngine as vge`. The import is the test itself (verifying the library can be imported), but the `vge` alias is not referenced in that method since the version is retrieved via `importlib.metadata`. The noqa comment suppresses the lint warning while preserving the PRD's intent of testing the import. Ruff format subsequently collapsed the `test_parse_case39` assertion onto a single line (within the 100-char limit).
