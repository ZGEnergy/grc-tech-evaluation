# Implementation Report: PRD 02/02 - Record-Type Mapping Guide

## Summary

Implemented the Record-Type Mapping Guide as a structured markdown document
at `data/fnm/docs/mapping-guide.md`. The document maps all 17 PSS/E v31
record types to tool-agnostic power-system abstractions and documents
native support across six evaluated tools (PyPSA, pandapower, GridCal,
PowerModels.jl, PowerSimulations.jl, MATPOWER).
