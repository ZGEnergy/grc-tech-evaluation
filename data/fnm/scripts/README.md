# FNM Scripts Directory

This directory contains Python modules for FNM data parsing, validation, and manifest
management.

## Contents

- `manifest_io.py` — Functions for loading, validating, and updating the FNM manifest
- `tests/` — Unit tests for all script modules

## Later Phases

- PSS/E RAW parser (Phase 1, D3)
- Supplemental CSV parsers (Phase 1, D4-D5)
- Validation scripts (Phase 1, D6)
