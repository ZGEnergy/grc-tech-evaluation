# FNM Reference Directory

This directory stores reference solutions and verification datasets used to validate
parser correctness (Phase 3). Reference data is compared against parser output to ensure
fidelity of the ingestion pipeline.

## Contents (populated by later phases)

- Known-good power flow solutions for test cases
- Reference bus/branch/generator data for spot-checking
- Verification datasets from independent sources

## NDA Notice

All files in this directory are derived from NDA-restricted FNM source data and must not
be committed to version control. The parent `.gitignore` blocks this entire directory.
