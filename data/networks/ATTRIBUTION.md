# Network Data Attribution

## IEEE 39-Bus Test Case (`case39.m`)

The New England 39-bus system, also known as the 10-generator, 39-bus system.
Originally described in:

> M. A. Pai, *Energy Function Analysis for Power System Stability*,
> Kluwer Academic Publishers, 1989.

Data sourced from the [MATPOWER](https://matpower.org/) repository
([GitHub](https://github.com/MATPOWER/matpower)), distributed under the
BSD 3-Clause License.

## ACTIVSg2000 (`case_ACTIVSg2000.m`)

Synthetic 2000-bus power system test case created by the
[Electric Grid Test Case Repository](https://electricgrids.engr.tamu.edu/)
at Texas A&M University.

> A. B. Birchfield, T. Xu, K. M. Gegner, K. S. Shetye, T. J. Overbye,
> "Grid Structural Characteristics as Validation Criteria for Synthetic Networks,"
> *IEEE Transactions on Power Systems*, vol. 32, no. 4, pp. 3258-3265, July 2017.

Licensed under [CC-BY-4.0](https://creativecommons.org/licenses/by/4.0/).
